# Revealed Cosmos Site v0.2 - GitHub Upload Instructions

This package is the next version of the live GitHub Pages site.

## What this update does
- Replaces placeholder portfolio cards with real samples.
- Adds selected visuals.
- Adds PDF sample links.
- Adds three service packages.
- Repositions the site toward hiring/outreach: worldbuilding, lore documentation, quest structure, and visual narrative proof.

## Upload steps using GitHub web UI
1. Open your `revealed-cosmos` GitHub repository.
2. Click `Add file` -> `Upload files`.
3. Drag ALL files from this package folder into GitHub, including:
   - `index.html`
   - all `.jpg` images
   - all `.pdf` documents
4. GitHub should show the files ready to upload.
5. Commit message: `Update portfolio site with real samples`
6. Click `Commit changes`.
7. Wait 1-3 minutes.
8. Refresh your live site:
   `https://themachinemuseum2026.github.io/revealed-cosmos/`

## Important
Do not upload private lore bibles, raw prompts, personal documents, banking/tax material, or full unpublished canon. Public site = public sample only.
